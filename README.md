# CRyPtO_VaULt 🔐

CRyPtO_VaULt is a Java-based security application developed for the Information Security course. The project focuses on protecting sensitive data using modern cryptographic techniques such as AES encryption and SHA hashing.

It allows users to securely encrypt and decrypt text or files while ensuring data integrity and confidentiality through strong encryption algorithms.

---

## 🚀 Features

* 🔒 AES-based text and file encryption
* 🔓 Secure decryption system
* 🛡️ SHA hashing for password and integrity verification
* 📂 File handling support
* 🖥️ User-friendly graphical interface
* ⚡ Fast and efficient encryption workflow
* 🔑 Secure key-based protection

## 🧠 Cryptographic Concepts Used

### AES (Advanced Encryption Standard)

AES is a symmetric encryption algorithm used to securely encrypt and decrypt sensitive data.

### SHA (Secure Hash Algorithm)

SHA hashing is used for:

* Password protection
* Data integrity verification
* Secure hashing operations
